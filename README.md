# Customer Segmentation & Spending Analysis

## Overview
This project analyzes the spending behaviour of 2,240 customers across six product categories using Microsoft Excel and Tableau Public. The goal was to identify key customer segments to help guide marketing decisions.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts)
- Tableau Public (Interactive Dashboard)

## Key Questions Answered
- Which education level spends the most?
- Which age group spends the most?
- Which marital status group spends the most?
- Which products are most popular?

## Key Findings
- Graduated customers are the highest spenders
- Customers aged 48–57 spend the most across all categories
- Married customers outspend all other groups
- Wine is the most popular product category by far
